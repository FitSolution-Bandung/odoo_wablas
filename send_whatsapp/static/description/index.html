MESSAGO - Send Quotation/Invoices to Whatsapp

MESSAGO adalah solusi integrasi yang memungkinkan Odoo mengirimkan Quotation dan Invoice secara otomatis ke WhatsApp pelanggan, tanpa perlu pengiriman manual. Untuk menggunakan fitur ini, pengguna perlu memiliki akun di layanan pihak ketiga, dan kami siap membantu dalam proses pendaftarannya agar integrasi berjalan lancar.

Fitur Addons

  • Send WhatsApp dari menu Quotation.
  • Send WhatsApp dari menu Request for Quotation.
  • Send WhatsApp dari menu Customer Invoices.
