Send Quotation/Invoices to Whatsapp

Addon "Send Quotation/Invoices to Whatsapp" adalah integrasi yang memungkinkan pengguna untuk mengirim pesan langsung melalui layanan WhatsApp menggunakan platform Odoo, dengan bantuan layanan pihak ketiga yang menyediakan API untuk interaksi dengan WhatsApp. Jadi pengguna harus memiliki akun di layanan pihak ketiga ini agar bisa menggunakan addon ini. Kami akan memberikan layanan bantuan untuk pendaftaran ke pihak ketiga ini.

Configuration

Fitur Addons

  • Send WhatsApp dari menu Quotation.
  • Send WhatsApp dari menu Request for Quotation.
  • Send WhatsApp dari menu Customer Invoices.
